Q) Write a java program to accept date of birth of a person and find out his age?

import java.time.LocalDate;
import java.time.Period;
class Example11 
{
	public static void main(String[] args) 
	{
		LocalDate dob = LocalDate.of(2002,8,15);
		LocalDate cur_date = LocalDate.now();
		
		Period p =Period.between(dob,cur_date);
		
		int age = p.getYears();
		System.out.println(age);
	}
}

input:

output:
	10:30:23



class Example13
{
	public static void main(String[] args) 
	{
		int totalSeconds = 37823;
		
		int hours = totalSeconds/3600;
		
		int minutes = (totalSeconds%3600)/60;
		
		int seconds = totalSeconds%60;
		
		System.out.println(hours+":"+minutes+":"+seconds);

}

Q) Sunita wakes up at 6:30 AM and reaches school at 7:15 AM.
Write a Java program to calculate total travel time in minutes.



class Example15
{
	public static void main(String[] args) 
	{
		int startHours = 6;
		int startMinutes = 30;
		
		int endHours = 7;
		int endMinutes = 15;
		
		int totalStartHours = (startHours*60) + startMinutes;
		int totalEndHours  = (endHours*60) + endMinutes;
		
		int travelTime = totalEndHours - totalStartHours;
		
		System.out.println(travelTime);
	
	}
}

Typescasting 
============
The process of converting from one datatype to another datatype is called typecasting.

In java, typecasting can be done in two ways.

1) Implicit typecasting 

2) Explicit typecasting 



1) Implicit typecasting
------------------------
If we want to store smaller value into a bigger variable then we need to use implicit typecasting.

A compiler is responsible to perform implicit typecasting.

There is no possibility to loss the information.

It is also known as widening or upcasting.

We can perform implicit typecasting as follow.

ex:
	byte	-->	short
				-->
					int -->	long  --> float	-->  double
				-->
			char

ex:
---

class Test
{
	public static void main(String[] args) 
	{
		byte b = 10;
		int i = b;
		System.out.println(i); //10
	}
}

ex:
---

class Test
{
	public static void main(String[] args) 
	{
		char ch = 'a';
		long l = ch;
		System.out.println(l); //97
	}
}

ex:
---

class Test
{
	public static void main(String[] args) 
	{
		int i=10;
		
		double d = i;
		
		System.out.println(d); // 10.0
	}
}



2) Explicit typecasting 
------------------------
If we want to store bigger value into a smaller variable then we need to use explicit typecasting. 

A programmer is responsible to perform explicit typecasting.

There is a possibility to loss the information.

It is also known as narrowing or downcasting.

We can perform explicit typecasting as follow.

ex:
	byte	<--	short
				<--
					int <--	long  <-- float	<--  double
				<--
			char


ex:
---

class Test
{
	public static void main(String[] args) 
	{
		double d = 10.56d;
		int i = (int)d;
		System.out.println(i); // 10
	}
}


ex:
---

class Test
{
	public static void main(String[] args) 
	{
		int i = 65;
		
		char ch = (char)i;
		
		System.out.println(ch); // A 
	}
}


ex:
---

class Test
{
	public static void main(String[] args) 
	{
		int i = 130;
		
		byte b = (byte)i;
		
		System.out.println(b); // -126
	}
}

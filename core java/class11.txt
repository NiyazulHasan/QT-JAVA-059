Main method 
===========
Our program contains main method or not. Either it is properly declared or not. It is not a responsibility of a compiler to check. It is a liability of a JVM to check for main method always at runtime. If JVM won't find main method then it will throw main method not found.

JVM always look for main method with following signature.

signature
---------
		public static void main(String[] args)

If we perform any changes in above signature then JVM will throw one runtime error 
called main method not found.

We can perform following changes in main method.

1) In traditional approach, order of modifiers is not important incase of public 
   static we can declare static public also.
   ex:
	static public void main(String[] args)

2) In traditional approach , We can replace String[] in following acceptable formats.
   ex:
	public static void main(String[] args)
	public static void main(String []args)
	public static void main(String args[])

3) In traditional approach , we can replace String[] with var-arg parameter.
   ex:
	public static void main(String... args)	


4) In traditional approach , we can change args with any java valid identifier.
   ex:
	public static void main(String[] args)


5) In traditional approach, main method will accept following modifiers.
   ex:
	synchronized
	strictfp 
	final 

6) Since Java 21, main method must not be public.
   ex:
	static void main(String[] args)

7) Since Java 21, we can declare instance main method.
   ex:
	void main(String[] args)


8) Since Java 21, String[] args is optional.
   ex:
	void main()



Q) Explain main method in java?

public
------
	JVM wants to call main method from anywhere. 

static 
------
	JVM wants to call main method without using object reference.

void 
-----
	Main method does not return anything to JVM.

main
-----
	It is a identifier given to a main method.

String[] args 
-------------
	It is a command line arguments.

Command Line Arguments 
======================
Arguments which are passing through command prompt such type of arguments are called command line arguments.

In command line arguments we need to pass our input values at runtime command.

ex:
	javac   Test.java
	
	java    Test  101  Alan  M  1000.0
		       |     |   |    |_____args[3]
		       |     |   |__________args[2]    
		       |     |______________args[1]
		       |____________________args[0]

ex:
---
class Test
{
	void main(String[] args)
	{
		System.out.println(args[0]);
		System.out.println(args[1]);
		System.out.println(args[2]);
		System.out.println(args[3]);
	}
}

We can use command line argument if we want calculate salary whenever we run the program.

ex:
---
class Test
{
	void main(String[] args)
	{
		//convert string to int 
		int basicSalary = Integer.parseInt(args[0]);
		int hra = basicSalary*10/100;	
		int da = basicSalary*20/100;
		int totalSalary = basicSalary + hra + da;
		System.out.println(totalSalary);	
	}
}

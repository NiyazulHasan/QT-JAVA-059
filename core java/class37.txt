Q) Write a java program display number digits , sum of digits and check it is even or odd?

input:
	NEX123

output:
	Number of digits : 3
	Sum of digits    : 6 
	It is even number 


class Test 
{
	public static void main(String[] args) 
	{
		String str="NEX123";
		
		String s = str.replaceAll("[^0-9]","");
		
		System.out.println("Number of digits :"+s.length());
		
		int sum = 0;
		for(char ch : s.toCharArray())
		{
			//convert character to number 
			int n = Character.getNumericValue(ch);
			sum+=n;
		}
		
		System.out.println("Sum of digits :"+sum);
		
		System.out.println((sum%2==0)?"It is even number":"It is odd number");
	}
}

Q) Write a java program to display the string in reverse order?

input:
	hello
output:
	olleh 


class Test 
{
	public static void main(String[] args) 
	{
		String str="hello";
		
		//reading reverse
		String rev="";
		for(int i=str.length()-1;i>=0;i--)
		{
			rev+=str.charAt(i);
		}
		
		System.out.println(rev);
	}
}

Q) Write a java program to check given string is palindrome or not?

input:
	racar 
output:
	It is a palindrome string 

class Test 
{
	public static void main(String[] args) 
	{
		String str="racar";
		
		//reading reverse
		String rev="";
		for(int i=str.length()-1;i>=0;i--)
		{
			rev+=str.charAt(i);
		}
		
		if(str.equals(rev))
			System.out.println("It is a palindrome string");
		else
			System.out.println("It is not a palindrome string");
			
	}
}

Q) Write a java program to find out maximum integer in a given string?

Input:
	1 kg of apple is 100 rupees and 2kg of mangoes 200 rupees

output:
	200


class Test 
{
	public static void main(String[] args) 
	{
		String str="1 kg of apple is 100 rupees and 2kg of mangoes 200 rupees";
		
		String[] sarr = str.split("\\D+");
		
		int max = Integer.MIN_VALUE;
		
		for(String s : sarr)
		{
			int n = Integer.parseInt(s);
			if(n>max)
			{
				max = n;
			}
		}
		System.out.println(max);
	}
}

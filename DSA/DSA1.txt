Linear Search 
=============
In linear search, we visit each element of an array eventually untill target element is found.

Linear search works for sorted and unsorted array.

In realtime we will use linear search to find out an object which is present in a collections.
ex:
	List<String>  list = List.of("apple","banana","orange");
	System.out.println(list.contains("banana")); // true 


Time Complexity 
--------------
	In Best Case, O(1) is a time complexity 
	ex:
		arr = 6 9 2 4 1 7 3 
		target = 6

	In Average Case, O(n) is a time complexity 
	ex:
		arr = 6 9 2 4 1 7 3 
		target = 4
	
	In Worst Case, O(n) is a time complexity 
	ex:
		arr = 6 9 2 4 1 7 3 
		target = 3
		or
		arr = 6 9 2 4 1 7 3 
		target = 10

Q) Write a java program to find out target element from given positive unsorted array?

Input:
	arr = 6 9 2 4 1 7 3 
	target = 4

Output:
	3


class LinearSearch 
{
	public static void main(String[] args) 
	{
		int[] arr ={6,9,2,4,1,7,3}; 
		int target = 4;
		int index = findIndex(arr,target);
		if(index!=-1)
			System.out.println("Element Found In "+index+" Index");
		else
			System.out.println("Element Not Found");
	}
	public static int findIndex(int[] arr,int target)
	{
		for(int i=0;i<arr.length;i++)
		{
			if(arr[i]==target)
			{
				return i;
			}
		}
		return -1;
	}
}

Q) Write a java program to find out number of occurance of a given target element?

input:
	arr = 6 7 2 7 1 7 5 6 7
	target = 7
output:
	4  


class LinearSearch 
{
	public static void main(String[] args) 
	{
		int[] arr = {6,7,2,7,1,7,5,6,7};
		int target = 7;
		int cnt = findTarget(arr,target);
		System.out.println(cnt);
	}
	public static int findTarget(int[] arr,int target)
	{
		int cnt = 0;
		for(int i : arr)
		{
			if(i==target)
			{
				cnt++;
			}
		}
		return cnt;
	}
}
